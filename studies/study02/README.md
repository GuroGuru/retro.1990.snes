### SnesDevStudies 2 - Loading and storing

This study shows how to **load registers** and **store values into memory**.

---

#### Topics introduced:

- Instructions:
    - [x] lda, ldx, ldy (loading)
    - [x] sta, stx, sty, stz (storing)

- Addressing modes:
    - [x] Immediate: #$c0
    - [x] Absolute: $c000

- Registers:
    - [x] Working registers (A, X and Y)
